Project Schedule Dopo aver analizzato la corretta sequenza delle attività, la loro durata, i relativi requisiti di risorse e vincoli di pianificazione, tutte le informazioni vengono finalizzate in una Shedulazione di Progetto o Project Schedule. Tale documento include le date di inizio e fine di ciascuna attività di progetto, la relazione tra le attività, le risorse necessarie, la durata compessiva del progetto, e così via.
