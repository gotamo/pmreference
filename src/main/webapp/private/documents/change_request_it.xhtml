Change Request Una richiesta di modifica (o Change Request) può essere definita come una qualsiasi richiesta inoltrata al Change Control Board (CCB) o a un qualsiasi altro gruppo di controllo delle modifiche. Queste richieste possono consistere in modifiche che vanno da modifiche minori, a modifiche significative che alterano drasticamente il progetto. Tuttavia, in genere le richieste di modifica formale riguardano modifiche più significative, mentre quelle meno significative sono apportate a livello di gestione del progetto. Inoltre in caso di necessità/urgenza viene dato al Project Manager l'autorità di approvare i change utilizzando meccanismi automatici del Change Control Board (CCB).
Una tipica change request può richiedere una modifica a un certo numero di aree del progetto, compresa l'espansione o riduzione dell'ambito del progetto, richieste di modifica di politiche, procedure, piani o processi, e le richieste di modifica delle spese e le richieste di aggiornamento della pianificazione.
Le richieste di modifica possono essere fatte direttamente o indirettamente e possono essere avviate internamente o esternamente. Una costante nella maggior parte dei processi di richiesta di modifica è che la richiesta deve essere formulata in modo formale, tramite una proposta scritta o un modulo di richiesta, e che le modifiche non devono essere implementate finché non vengono formalmente approvate.
Le richieste di modifica possono essere approvate, rigettate o sospese. Le possibili azioni generate dal processo di approvazione delle modifiche sono:
  • Azione correttiva - applicata nel caso di un problema che si è verificato ed è finalizzata a riallineare le prestazioni del progetto con il piano di Project Management.
  • Azione preventiva - applicata nel caso in cui il problema non si è ancora verificato ed è finalizzata a garantire che l'andamento futuro del progetto sia allineato con il piano di project Management.
  • Correzione di un difetto - attività volta a modificare un prodotto o un componente non conforme alle specifiche.
  • Aggiornamenti - modifiche a documenti o piani di progetto sottoposti a controllo formale.
